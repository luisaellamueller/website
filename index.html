Hi, I am Luisa! I am a curious computer scientist with a background in software engineering and data science. I am currently studying at ETH Zurich and completed my bachelors at HSG.

I believe in the power of technology to transform ideas into reality, and I'm constantly exploring new ways to solve complex problems through innovative solutions.

Projects

Screentime reduction App

A cross-platform application designed to help children reduce screen time by offering a voucher-based incentive system. Developed with support for both native iOS and Android features to ensure a smooth and responsive user experience.

Pet Sitting Application

A matchmaking platform that connects pet owners with trusted pet sitters. Find close petsitters by map searching and rate the petsitter. Built using Java and Spring Boot, with a Vaadin-based frontend for a responsive and modern user interface.

Movie Match

A smart movie matchmaking app that helps users discover films they'll love. Leveraging a Hugging Face NLP model, the app provides intelligent search and personalized recommendations based on user preferences.
